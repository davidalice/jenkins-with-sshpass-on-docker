# jenkins-with-sshpass-on-docker

## Tools

- [Jenkins](https://jenkins.io/)
- With
  - sshpass

## How to build image

- clone the repository

```
  $ git clone https://github.com/kiyohome/jenkins-with-sshpass-on-docker.git
```

- build the image

- no proxy

```
  $ docker build -t kiyohome/jenkins-with-sshpass .
```
